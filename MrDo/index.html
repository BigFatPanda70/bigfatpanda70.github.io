Mr Do!

Click here to play

Note: This software is a personal project to recreate a game I loved to play in the arcades as a kid. It is an attempt to create a playable reproduction of the hardware and software. It uses the original ROMS from the arcade machine, so it is not for sale or general distribution The roms have not been released into the public domain and I don't have any rights regarding them. (I've asked.. but as yet had no reply.).

The emulator is not 100% perfect, I still have some issues with colour palettes and some hardware screen scrolling issues but it is mostly playable.

Controls:

Keyboard:
arrow keys to move
5 to insert coins
1 to start player one game
2 to start player two game
s to throw ball

Joystick Support:
Most standard joysticks should work (including PS4 controllers).

Touchscreens
Right hand side of the screen : throw ball.
Left hand side: swipe and hold to move up,down,left and right.
Use the hamburger menu icon (grey, 3 lines top right) to add coins and start the game.
