Base64 Encoder

Web browsers have a lot of security features built in, which made it impossible to load images/textures without turning off security features. To get around the cross origin resource sharing restrictions (CORS) when testing locally, I found base64 encoding images to be a workable solution

I've uploaded it here in case anyone else finds it useful. Base64 Encoder Click here
