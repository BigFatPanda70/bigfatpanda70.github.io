28th April 2020

Where has the month gone ?

I've been working on adding hardware support for the bat'n' ball game so, where available, it will use 'webgl' which should hopefully make the game play faster on some hardware setups.

I've also uploaded a base64 image encoder which may be useful to someone for creating embedded image data (see links section).

4th April 2020:

My "Bat n ball" game is still very much a work in progress, but as of today it's playable and
since we're all stuck in lockdown I've uploaded it to help try and relieve the boredom (for 5 minutes at least).

It runs in a browser, so there is nothing to download and install. I've tested it on android, IOS, Linux and Windows so I'm fairly confident it will work.

Enjoy.

-Nick.

Controls:


Keyboard:

Move left : key A,J, or left arrow key..

Move right : key D,L, or right arrow key..

If you are lucky enough to have a keyboard press 'P' to pause for a loo break.


Touch screens:

Move left : press the left side of the screen

Move right : press the right side of the screen


Joystick support:

Is available for the Firefox web browser.
